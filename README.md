# 🐉 DnD Bot — створення персонажа D&D 5e (2014)

Telegram **Mini App**, який веде тебе по створенню персонажа
(два режими — «Повний» і «Швидкий»), а наприкінці **генерує PDF-чарник
прямо у браузері** й зберігає його на пристрій. **Сервер не потрібен** —
достатньо статичного хостингу (GitHub Pages).

## Як це працює

```
Telegram  ──Menu Button──▶  відкривається Mini App
                            │
                            ▼
             Mini App (webapp/ на GitHub Pages)
             • список рас → сторінка раси → список класів → сторінка класу
             • повний чарник із передзаповненими расою й класом
             • живий підрахунок модифікаторів, HP, AC, DC…
             • екран-перевірка
                            │  Готово
                            ▼
             PDF малюється в браузері (jsPDF) → share / завантаження на пристрій
```

Уся логіка D&D **і генерація PDF** — на клієнті. Жодного бекенду в рантаймі:
Telegram лише показує кнопку, що відкриває сторінку на GitHub Pages.

> ℹ️ **PDF не приходить повідомленням у чат** — Mini App не може покласти файл
> у чат без запущеного бота. Натомість PDF зберігається/розшарюється на пристрій
> (share-меню ОС, з fallback на завантаження файлу).

## Структура

| Файл | Призначення |
|------|-------------|
| `webapp/index.html` | Уся Mini App: розмітка + стилі + дані D&D + логіка створення персонажа |
| `webapp/pdf.js` | Генерація PDF-чарника у браузері (jsPDF) — малює власний український бланк |
| `webapp/vendor/jspdf.umd.min.js` | Бібліотека jsPDF |
| `webapp/vendor/fonts.js` | Кириличний шрифт (DejaVu Sans, засабсечено) для jsPDF |
| `bot.py`, `pdf_generator.py`, `requirements.txt` | **Необов'язково / legacy** — стара версія з ботом, що надсилав PDF у чат. Для нового флоу не потрібні |

> Для деплою достатньо папки **`webapp/`** — це вся робоча апка. Python-частина
> лишена лише як довідка про попередній варіант.

## Деплой (без сервера)

### 1. Викласти на GitHub Pages

1. Залий проєкт у **публічний** репозиторій на GitHub.
2. **Settings → Pages → Build and deployment → Source: `Deploy from a branch`**,
   гілка `main`, папка `/ (root)`.
3. За хвилину апка буде доступна за адресою:
   `https://<твій-нік>.github.io/<репо>/webapp/`
4. Перевір, що вона відкривається у браузері (кнопка «Готово» завантажить PDF).

> Telegram вимагає **HTTPS** — GitHub Pages його дає автоматично.
> Приватні репо на безкоштовному тарифі Pages не роздають — тоді бери
> **Cloudflare Pages** (працює і для приватних).

### 2. Підключити як Mini App у Telegram

Бот **не запускається** — потрібен лише щоб показати кнопку, що відкриває апку:

1. У [@BotFather](https://t.me/BotFather): `/newbot` (або взяти наявного).
2. `/mybots` → обрати бота → **Bot Settings → Menu Button → Configure menu button**.
3. Вказати URL апки (`https://<нік>.github.io/<репо>/webapp/`) і назву кнопки.
4. Відкрити бота в Telegram → тапнути кнопку меню → відкриється майстер персонажа.

## PDF-чарник

Генерується **власний український бланк** (повністю малюється кодом у браузері,
`webapp/pdf.js` на базі jsPDF). Заповнюються всі поля: 6 характеристик із модифікаторами, бонус
майстерності, 6 рятівних кидків, **усі 18 навичок**, бойові показники (КБ,
ініціатива, швидкість), HP і кості здоров'я, спорядження, риси, володіння та
мови, нотатки. Для заклиначів додається окрема **сторінка заклять** із шапкою
(характеристика, DC рятівного кидка, бонус на влучання) — сам список заклять
лишається порожнім (закляття у флоу не обираємо).

Риси показуються з **коротким описом** (назва жирним + пояснення нижче);
маркери майстерності: ● — володіє, ○ — ні. Рядки з майстерністю підсвічені.

Файл називається `<Назва персонажа>_<раса>_<ім'я гравця>.pdf`.

## Перевірка локально

Mini App — звичайна статика, тож просто:

```bash
python -m http.server 8099 --directory webapp
```

і відкрий `http://localhost:8099/`. Пройди флоу до «Готово» — PDF згенерується
й завантажиться (share-меню доступне лише в захищеному контексті / на пристрої).

> Шрифт для jsPDF (`webapp/vendor/fonts.js`) отримано сабсетингом DejaVu Sans
> під потрібні гліфи (латиниця + кирилиця + символи ● ○ • —). Важливо **не**
> видаляти при сабсетингу таблицю `name` — інакше парсер шрифту в jsPDF ламається.

## Що підтримується

- **Флоу з браузингом**: список рас (плитки) → сторінка раси (портрет, бонуси,
  особливості, підраси на вибір, рекомендації класів) → список класів →
  сторінка класу (**вибір рівня**, HP на обраному рівні, **що отримуєш на кожному
  рівні** — здібності з описами, вибір підкласу на його рівні, комірки заклять /
  Ці / лють тощо) → повний чарник із **передзаповненими** расою й класом → PDF.
  Скрізь є кнопка «назад» угорі зліва.
- **Попап підвищення/черти**: на рівнях 4/8/… з'являється вікно «характеристики
  чи черта»; черта → окремий екран вибору черти (з описами й +1, якщо напів-черта).
- **50 рас** із усіх книг (PHB, VGM, EEPC, GGR, MOT, TP, MTF, ERLW, WBW, SCC, SAS, AI, OGA, SDQ, LR)
  з **помітками джерела** та **фільтром-чіпами за редакцією** (типово всі, тап — лише обрані);
  підраси теж із інших книг (SCAG/MToF/EGW). SVG-портрети, рекомендації класів
- **12 класів**, кожен із підкласами (PHB + по одному з XGE), усі з описом
- **Опційні вміння класів (Tasha)** — чекбокс перед вибором раси (типово увімкнено),
  позначаються «опц.» і потрапляють у чарник
- **18 навичок** зі зрозумілими назвами (Магія = Arcana); **8 передісторій**, 9 світоглядів
- **Англійські назви** рас і класів; ресурси по рівнях (комірки заклять, Лють,
  Ці, Підступна атака…) винесено в окремий блок із поясненням, що це
- Кожен вибір (раса, підклас, черта, передісторія) має іконку ⓘ з описом
- Вибір **характеристики або черти** на рівнях 4/8/12/16/19 (+ клас-специфічні),
  стартова черта для варіанта людини
- Черти: опис + автозастосування числових ефектів (+1 характеристика,
  +2 HP/рівень «Витривалий», +5 ініціатива «Насторожений», рятівний «Стійкий» тощо)
- Навички згруповані **за джерелом** (раса / клас / передісторія) з підсвічуванням
  дублів; рятівні кидки — за джерелом
- Характеристики: стандартний набір / point buy (27) / кидки 4d6 / вручну
- Автопідрахунок: модифікатори, бонус майстерності, HP, AC, ініціатива,
  рятівні кидки, навички, пасивна уважність, DC та бонус атаки закляттям
- Заклиначі отримують окрему сторінку для заклять
